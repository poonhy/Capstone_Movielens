# Capstone_Movielens
R capstone project on Movielens dataset

The 3 files in this repository are the submissions for the Movielens capstone project for the "Data Science: Capstone" course on edx.

The 3 files are:
i)    Movielens.rmd which is the report for the project in rmd format;
ii)   Movielens.pdf which is the report in pdf form; and
iii)  Movielens.R containing all the R code for the project.
